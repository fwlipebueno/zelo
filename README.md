## English

### About the project
Zelo is a front-end web project developed for the Web Front-End Programming course, focused on SDG 3 — Good Health and Well-Being.

The project is an informative and community-oriented website designed to promote prevention, healthy habits, public health campaigns and useful well-being resources. It is not a medical system, diagnosis tool, telemedicine platform or treatment service.

### Purpose
- Promote accessible information about health and well-being.
- Encourage prevention and healthy habits.
- Connect the community with useful resources and campaigns.
- Practice semantic HTML and CSS in a real web project context.

### Target audience
The project is intended for local community members, students, workers, families and people interested in prevention, health education and well-being.

### Sustainable Development Goal
SDG 3 — Good Health and Well-Being.

Zelo is aligned with the goal of promoting healthy lives and well-being for people of all ages through accessible information and community awareness.

### Features implemented in Project 1
- Home page for the proposed solution.
- Registration page with a form containing more than five fields.
- Login page with e-mail and password fields.
- Navigation menu.
- Semantic HTML structure.
- Responsive CSS layout.
- Local SVG images.
- Useful links related to health, well-being and SDG.
- Footer with links to each team member's résumé.
- Visual identity focused on clarity, trust and accessibility.

### Technologies
- HTML5
- CSS3
- Google Fonts
- SVG assets
- GitHub Pages

## Português

### Sobre o projeto
Zelo é um projeto web front-end desenvolvido para a disciplina Programação Web Front-End, com foco no ODS 3 — Saúde e Bem-Estar.

O projeto é uma página web informativa e comunitária criada para promover prevenção, hábitos saudáveis, campanhas de saúde pública e recursos úteis de bem-estar. Ele não é um sistema médico, ferramenta de diagnóstico, plataforma de telemedicina ou serviço de tratamento.

### Objetivo
- Promover informação acessível sobre saúde e bem-estar.
- Incentivar prevenção e hábitos saudáveis.
- Aproximar a comunidade de recursos e campanhas úteis.
- Aplicar HTML semântico e CSS em um contexto prático de desenvolvimento web.

### Público-alvo
O projeto é voltado para moradores da comunidade local, estudantes, trabalhadores, famílias e pessoas interessadas em prevenção, educação em saúde e bem-estar.

### Objetivo de Desenvolvimento Sustentável
ODS 3 — Saúde e Bem-Estar.

O Zelo está alinhado à promoção de uma vida saudável e do bem-estar para pessoas de todas as idades por meio de informação acessível e conscientização comunitária.

### Funcionalidades implementadas no Projeto 1
- Página principal referente à solução proposta.
- Página de cadastro com formulário contendo mais de cinco campos.
- Página de login com campos de e-mail e senha.
- Menu de navegação.
- Estrutura HTML semântica.
- Layout responsivo com CSS.
- Imagens SVG locais.
- Links úteis relacionados à saúde, bem-estar e ODS.
- Rodapé com links para o currículo de cada integrante.
- Identidade visual focada em clareza, confiança e acessibilidade.

### Tecnologias utilizadas
- HTML5

- Google Fonts

- GitHub Pages
